## 🌟 11. Weekly Learning Habits & Community
*Stay updated and build your professional network.*

### 📰 Recommended Sources
- **Engineering Blogs:** OpenAI Research, Anthropic Engineering, Hugging Face Blog, AWS Machine Learning Blog.
- **Newsletters:** The Batch (DeepLearning.AI), Import AI, Last Week in AI.
- **Communities:** r/MachineLearning, LangChain Discord, Hugging Face Forums, local AI meetups.
- **Podcasts:** TWIML AI, Data Science at Home, Machine Learning Street Talk.

### 🔄 Weekly Action Items
- [ ] **Read 1 AI Paper:** Start with "Attention is All You Need", "LoRA", or recent ArXiv preprints.
- [ ] **Read 3 Engineering Blogs:** Focus on release notes and production war stories.
- [ ] **Reproduce 1 Open-Source Project:** Clone a repo, run it locally, and understand its structure.
- [ ] **Make 1 Open-Source Contribution:** Fix a typo, improve docs, or add a test to `langchain`, `transformers`, or `prometheus`.
